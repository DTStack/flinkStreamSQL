## 1.语法
```
 CREATE VIEW viewName
        [ (columnName[ , columnName]*) ]
            AS queryStatement;
 或
 CREATE VIEW viewName [ (columnName[ , columnName]*) ];
 INSERT INTO viewName queryStatement;
```
