## 介绍
 * 基于开源的flink，对其实时sql进行扩展   
    * 自定义create table 语法（包括源表,输出表,维表）
    * 自定义create view 语法
    * 自定义create function 语法
    * 实现了流与维表的join
    * 支持原生FLinkSQL所有的语法
    * 扩展了输入和输出的性能指标到promethus
